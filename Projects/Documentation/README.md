# How to Create Animated Diagrams for Documentation  

This guide will help you create animated diagrams similar to those commonly seen on LinkedIn. Below are three methods using Excalidraw, Figma, and Canva.  

## Option 1: Excalidraw + OBS Studio (Hand-Drawn Style)  
### Steps:  
1. **Create the Diagram in Excalidraw**  
   - Visit [Excalidraw](https://excalidraw.com/) and design your diagram.  
   - Use the built-in "hand-drawn" style for a casual, sketched look.  
   - Export your work as PNG/SVG.  

2. **Record Animation with OBS Studio**  
   - Download and install [OBS Studio](https://obsproject.com/).  
   - Set up a **screen recording** scene with your Excalidraw window.  
   - Slowly reveal elements (e.g., dragging them in or erasing sections live).  
   - Record and export as a video.  

3. **Convert to GIF (Optional)**  
   - Use [Ezgif](https://ezgif.com/video-to-gif) or Photoshop to convert your video to a GIF.  

---

## Option 2: Figma Smart Animate (For Smooth Transitions)  
### Steps:  
1. **Create the Diagram in Figma**  
   - Open [Figma](https://www.figma.com/).  
   - Design your diagram using text, icons, and shapes.  
   - Use layers to separate different elements.  

2. **Animate with Smart Animate**  
   - Duplicate your diagram into multiple frames.  
   - Move elements slightly or change opacity in each frame.  
   - Use **Smart Animate** between frames for smooth transitions.  

3. **Export as GIF or MP4**  
   - Use **Prototype Mode** to preview animations.  
   - Export as GIF or MP4 via [Figma Plugins](https://www.figma.com/community/plugins).  

---

## Option 3: Canva (For Simple Infographics)  
### Steps:  
1. **Design in Canva**  
   - Use [Canva](https://www.canva.com/) with a presentation or infographic template.  
   - Add icons and text.  

2. **Animate Elements**  
   - Click on an element, then select "Animate" in the toolbar.  
   - Use effects like "Fade In" or "Rise."  

3. **Export as GIF or MP4**  
   - Save your animation as a **GIF** or **MP4** for easy sharing.  

---

## Which One Should You Use?  
| Tool | Best For |
|------|---------|
| **Excalidraw + OBS** | Hand-drawn, casual animations |
| **Figma** | Professional UI-like animations |
| **Canva** | Quick and easy animations |
